Proyecto desarrollado para el Introducción a la Programación 
Tecnológico de Costa Rica - I Semestre 2026

# Descripción del Proyecto
Videojuego inspirado en pokemon utilizando python y Tkinter para la interfaz gráfica, el jugador inicia ingresando su nombre y seleccionando un avatar (5 opciones diferentes) y luego escoje 3 pokemons para formar su equipo.
El jugador debe seleccionar un pokemon de su equipo para que se enfrente a un pokemon del enemigo elegido al azar, durante el combate los pokemons se turnan para atacar al otro, cuando la vida de uno llega a 0, este revive y pasa a pertenecer al equipo contrario. Este ciclo se repite hasta que uno de los equipos se quede sin pokemons.

# Requerimientos Cumplidos

- Pantalla inicial con ingreso de nombre, selección de 3 Pokémon y avatar
- Sistema de batalla contra rival aleatorio
- 10 Pokémon con atributos (vida, ataque y defensa)
- Sistema de daño: daño = ataque - defensa
- Captura de Pokémon con restauración de vida
- Puntaje por cada Pokémon capturado
- Pantalla de Mejores Puntajes (Top 10 persistente con JSON)
- Interfaz gráfica completa con Tkinter

# Tecnologías Utilizadas

- Python 3
- Tkinter (biblioteca estándar para GUI)
- Pillow (para el manejo de imágenes)
- JSON (para guardar los mejores puntajes)
